## Dictionary App
A simple and interactive web application that allows users to look up word meanings, synonyms, and pronunciation audio using the Free Dictionary API.
 Features
Search for any English word
View definitions
View synonyms
Listen to pronunciation audio
Error message when a word is not found
Simple and responsive UI


## API Used
This project uses the Free Dictionary API:

https://api.dictionaryapi.dev/api/v2/entries/en/<word>
Example:

https://api.dictionaryapi.dev/api/v2/entries/en/technology
